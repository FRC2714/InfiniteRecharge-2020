# InfiniteRecharge-2020

Robot code for Team 2714s Robot designed to play Infinite Recharge. 

![Robot Demo GIF](./RobotGIF.gif)


## Build
Use [gradle](https://gradle.org/) to build and deploy.

```bash
./gradlew build
```

## Deploy

```bash
./gradlew deploy
```
